## üîë Understanding D&I Funding Models

**Two Primary Models:**

### 1. Direct Individual Support
Traditional fellowships and scholarships where individuals apply directly:
- **Examples**: AAUW Fellowships, Point Foundation, NFB Scholarships
- **Strategy**: Browse categories above and apply directly

### 2. Institutional & Programmatic Grants
Large awards to universities that create internal support programs:
- **Examples**: Clare Boothe Luce Program, NASA MUREP, McNair Scholars
- **Strategy**: Identify universities with these grants for supportive environments
