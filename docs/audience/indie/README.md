## Mid-Range Grants

**Domains**: Open Tech / OSS / Internet, Computer Science & AI

Funding for public interest technology projects

### [Mozilla Grants](https://foundation.mozilla.org/en/what-we-fund/)

**Organization**: Mozilla Foundation | **Amount**: $10,000-$50,000 (Common Awards) | **Deadline**: Varies by program

**Domains**: Open Tech / OSS / Internet, Computer Science & AI

Support for open internet and privacy projects

### [Open Technology Fund](https://www.opentech.fund/)

**Organization**: OTF | **Amount**: Variable (typically $10,000-$900,000) | **Deadline**: Rolling applications

**Domains**: Open Tech / OSS / Internet

Supporting open technologies for global internet freedom

### [NLnet Foundation Grants](https://nlnet.nl/propose/)

**Organization**: NLnet Foundation | **Amount**: €5,000-€50,000 | **Deadline**: Rolling (8 calls per year)

**Domains**: Open Tech / OSS / Internet

Supporting open source projects contributing to open internet

### [Emergent Ventures](https://www.mercatus.org/emergent-ventures)

**Domains**: Computer Science & AI, Social Sciences & Policy, Open Tech / OSS / Internet

Supporting innovative ideas and moonshot projects

### [Fast Grants](https://fastgrants.org/)

**Organization**: Fast Grants | **Amount**: $10,000-$500,000 | **Deadline**: Rolling (rapid decisions)

**Domains**: Life Sciences & Biomedicine

Rapid funding for COVID and science projects (varies by availability)
